🚀 GitHub Pages Deployment

1. Upload files

Upload index.html + .github/workflows/deploy.yml to GitHub repo.

2. Add Secret

Settings → Secrets → Actions → New secret
Name: GEMINI_API_KEY
Value: your AIzaSy… key

3. Enable Pages

Settings → Pages → Source: GitHub Actions
Push to main → auto deploys!

4. You're live!

URL: https://USERNAME.github.io/REPO/
Key injected securely at build time.

🔒 Security: deploy.yml uses sed to replace REPLACE_ME_WITH_KEY in index.html with your GitHub secret. Key never stored in code.
