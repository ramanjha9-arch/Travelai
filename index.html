⚙️ How to Activate Gemini AI

Step 1

Go to aistudio.google.com/app/apikey and create a free Gemini API key. It takes 30 seconds.

Step 2

Paste the key (starts with AIzaSy…) into the orange bar at the very top of this page. Click Activate AI.

🔒 Your key is stored only in your browser's localStorage — never sent to any server other than Google's Gemini API. You can clear it anytime with the Clear button.

🕸️ Web Crawl Fallback

No API key? The app automatically switches to Web Crawl Mode — fetching live data from Wikipedia, REST Countries, and Exchange Rate APIs to build your itinerary. Fully functional, just less personalised.

Fallback chain: Gemini 2.5 Flash → Gemini 1.5 Flash → Web Crawl (Wikipedia + REST APIs) → Smart offline templates
