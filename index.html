🚀 Deploy on GitHub Pages (Free)

Step 1 — Upload to GitHub

Upload index.html and .github/workflows/deploy.yml to a new GitHub repo.

Step 2 — Add GitHub Secret

Go to Settings → Secrets → Actions → Add secret named GEMINI_API_KEY with your key value.

Step 3 — Enable Pages

Go to Settings → Pages → Source: GitHub Actions. Push to main → auto-deploys!

Step 4 — Live!

Your site: https://USERNAME.github.io/REPO/ — API key auto-injected from secret.

🔒 Security: The deploy.yml uses sed to replace REPLACE_ME_WITH_KEY placeholder in index.html with the GitHub secret at build time. Key is never stored in your repo.
